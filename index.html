Healthcare · Python · Power BI

COVID-19 Mortality Dashboard

⚠ Problem

Healthcare institutions lacked a clear visual framework to identify which comorbidities — diabetes, hypertension, cardiovascular disease — most significantly amplified COVID-19 mortality risk.

⚙ Solution

Built a Python-powered ETL pipeline to ingest and clean multi-source patient datasets, then developed an interactive Power BI dashboard with comorbidity cross-filters and mortality heatmaps.

Result: Surfaced that patients with 3+ comorbidities carried a significantly elevated mortality risk — visualised in a single-page dashboard consumable by non-technical stakeholders.

ML · Regression · PakWheels

Car Price Prediction Engine

⚠ Problem

Pakistan's used-car market (dominated by PakWheels) suffers from extreme price opacity — buyers and sellers have no reliable benchmark, leading to systematic mispricing.

⚙ Solution

Scraped and engineered a feature-rich dataset from PakWheels listings. Applied Linear Regression with feature selection, outlier handling, and cross-validation to build a locally calibrated valuation model.

Result: Achieved 94.17% prediction accuracy — a model deployable as a browser extension or API that gives users a fair-price estimate before they negotiate.
Python Scikit-learn Regression Feature Eng. PakWheels
🛒
E-commerce · RFM · UCI Dataset

Online Retail Insights

⚠ Problem

A raw transactional dataset from the UCI ML Repository was riddled with duplicates, nulls, and cancelled orders — making meaningful customer segmentation impossible without rigorous preprocessing.

⚙ Solution

Performed a 7-stage data cleaning pipeline, then executed RFM (Recency, Frequency, Monetary) analysis to segment customers into strategic tiers: Champions, At-Risk, and Dormant.

Result: Identified top 15% of customers generating ~60% of revenue — enabling targeted retention strategies with clear, visual segment profiles ready for marketing activation.
Python Pandas RFM Analysis Matplotlib UCI ML
